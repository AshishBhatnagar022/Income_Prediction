# Income_Prediction
<b>Predict wheather income exceeds $50K/yr based on census data</b>

Fields in the dataset include:

* Age
* Workclass
* Education
* Education.num
* Marital Status
* Occupation
* Relationship
* Race
* Sex
* Capital_Gain
* Capital_Loss
* House_per_week
* Native_Country
* Income

This project is carried out on dataset freely avaliable on Kaggle
